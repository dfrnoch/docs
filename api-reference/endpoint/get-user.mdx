## Get User Details

Retrieve detailed information about a user including their pricing configuration and order history.

### Endpoint

<ParamField path="id" type="string" required>
  The unique identifier of the user
</ParamField>

```bash
GET https://tronspace.vip/api/public/user/{id}
```

### Authentication

<Note>
  Not required
</Note>

### Response

<ResponseField name="id" type="string">
  Unique identifier of the user
</ResponseField>

<ResponseField name="name" type="string">
  User's display name
</ResponseField>

<ResponseField name="status" type="enum">
  Current user status (ACTIVE, INACTIVE)
</ResponseField>

<ResponseField name="wallet" type="string">
  TRON wallet address
</ResponseField>

<ResponseField name="priceFor5Min" type="number">
  Price rate for 5 minute duration
</ResponseField>

<ResponseField name="priceFor15Min" type="number">
  Price rate for 15 minute duration
</ResponseField>

<ResponseField name="priceFor1H" type="number">
  Price rate for 1 hour duration
</ResponseField>

<ResponseField name="orders" type="array">
  List of user's orders with transaction details
</ResponseField>
